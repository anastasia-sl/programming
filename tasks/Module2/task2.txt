Написать функцию, которая будет переворачивать массив.

Алгоритм работы функцииЖ
1. На вход принимается объект, которые необходимо проверить, является ли он массивом.
Если не является, то сгененрировать ошибку с сообщением "Argument is not a Array";
2. объявить новый массив, в который будут наполняться значения
3. объявить цикл for со счетчиком, который изначально равен длинне массива минус 1. Условие выполнения цикла должно быть
только если счетчик больше или равно 0, каждуй шаг счетчик уменьшается на 1;
4. внутри цикла нужно брать текущий элемент из исходного массива и вносить (пушить) его в новый массив
5. по завершении вернуть массив из функции

написать вывод в консоль двух массивов - исходный и перевернутый
